\# 🔥 HSE Heat Stress Alert System

> Real-time monitoring \& WhatsApp alerts for worker safety. Created \& Licensed by \*\*Nazeer Kadhar, HSE Professional\*\*.



\## 📋 Overview

A professional-grade web application designed for HSE professionals to monitor real-time heat stress conditions, calculate WBGT \& Heat Index, and receive automated safety alerts. Built for field safety teams, construction sites, and industrial operations.



\## ✨ Current Features

\- 🌍 Real-time weather data integration (OpenWeatherMap)

\- 📊 Automated Heat Index \& WBGT calculations

\- 🚦 5-Level Risk Assessment (Safe → Extreme)

\- 📱 Subscription-based WhatsApp alert system

\- 🎨 Responsive, mobile-friendly dashboard

\- ⚡ Auto-refresh \& scheduled monitoring (every 30 mins)

\- 📋 HSE-compliant safety recommendations per risk level

\- 🔒 Professional branding \& statutory warnings



\## 🛠️ Tech Stack

\- \*\*Backend:\*\* Node.js, Express.js, Axios, node-cron, dotenv

\- \*\*Frontend:\*\* Vanilla HTML5, CSS3, JavaScript (ES6+)

\- \*\*APIs:\*\* OpenWeatherMap, Twilio (WhatsApp)

\- \*\*Deployment:\*\* Render.com (Cloud)



\## 🌐 How to Use

1\. Visit: `https://heatindex-nazeerkadhar.onrender.com`

2\. Enter any city name

3\. Click \*\*Check Heat Stress\*\* to view real-time metrics

4\. Subscribe with your WhatsApp number for automated alerts

\*(No installation required. Works on any browser or mobile device.)\*
